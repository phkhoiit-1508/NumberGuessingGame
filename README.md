# \# Number Guessing Game (C++)

# 

# A console-based number guessing game written in C++.

# 

# \## Features

# \- Classic guessing mode

# \- Limited attempts mode with difficulty levels

# \- Reverse guessing mode (computer guesses using binary search)

# 

# \## Technologies

# \- Language: C++

# \- Concepts: loops, function, conditionals, binary search

# 

# \## How to Run

# 1\. Compile the program:

# &nbsp;  g++ main.cpp -o game

# 2\. Run:

# &nbsp;  ./game

#
